Todo
=========

Simple todo app with jquery and php

Requirements
----

- Database called 'todo'
- Table called 'tasks'

Specify database settings in ```settings.php```

#### Table structure
```
+--------+----------+--------------------+-----------+
| taskId | taskName | taskExpirationDate | completed |
+--------+----------+--------------------+-----------+
```
